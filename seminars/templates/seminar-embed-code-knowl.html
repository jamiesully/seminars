Include the HTML code below on your webpage to display the series schedule there. The code consists of a <div> tag and a <script> tag.

To include the schedules of multiple series, you need the <div> tag for each but the <script> tag only once, so copy the code block for each series to your webpage and remove all but one copy of the <script> tag.

By default, the code uses our styling when displaying the schedule. If instead you want your webpage's style settings to apply, then after copying the code, change 'addCSS': true to 'addCSS': false inside the <script> tag.

The bottom of each schedule will include a footer saying "Powered by {{topdomain}}". To remove this, delete the word sitefooter inside the <div> tag.
